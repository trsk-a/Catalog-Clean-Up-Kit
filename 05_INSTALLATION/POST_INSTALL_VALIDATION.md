# Post-Install Validation

## Pass Criteria
- Main workflow opens without missing node errors.
- Preset files are readable and applied.
- One in-scope sample runs to completion.
- Output is exportable in at least one ratio.

## Fail Codes
- PI-01 missing nodes
- PI-02 missing models
- PI-03 OOM on valid preset
- PI-04 malformed workflow template
- PI-05 output invalid against checklist
