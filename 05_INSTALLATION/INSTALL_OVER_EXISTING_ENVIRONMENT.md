# Install Over Existing Environment (Limited Support)

## Warning
This path has higher break risk and is not the primary support path.

## Safe Procedure
1. Backup your existing ComfyUI folder.
2. Compare your current nodes/models against compatibility matrix.
3. Remove or isolate conflicting node forks.
4. Add only pinned dependencies needed by CCKU.
5. Run smoke workflow.

## Disclaimers
- No guarantee for heavily customized stacks.
- If conflicts persist, migrate to clean install.
