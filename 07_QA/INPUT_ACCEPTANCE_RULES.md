# Input Acceptance Rules

## Minimum
- JPG/PNG/WebP
- Long edge >= 1024 px
- Product visibly readable

## Recommended
- Long edge 1600-3000 px
- Simple or recoverable background
- Product centered or mostly visible

## Reject Conditions
- Extreme transparency complexity
- Mirror-grade reflectives as critical case
- Fashion on-body
- Critical microtext fidelity required
