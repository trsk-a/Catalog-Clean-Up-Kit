# Start Here

This package is a narrow commercial toolkit for product photo clean-up (Catalog Clean-Up Kit V1).

## First 5 Actions
1. Read `WHAT_THIS_PRODUCT_IS_AND_IS_NOT.md`.
2. Read `../04_DEPENDENCIES/COMPATIBILITY_MATRIX.md`.
3. Follow clean install guide for your OS in `../05_INSTALLATION`.
4. Run post-install validation.
5. Launch Quick Start and process one in-scope sample image.
