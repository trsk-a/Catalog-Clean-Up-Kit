# Scope Freeze - Catalog Clean-Up Kit V1

## One-Sentence Decision
Catalog Clean-Up Kit V1 is launched first as a narrow kit for cleaning and improving rigid/semi-rigid product photos for catalog, e-commerce, marketplace, and static social outputs.

## Mandatory Scope Rules
1. V1 solves clean-up + neutral background + export readiness only.
2. V1 optimizes for stability, reproducibility, and low support load.
3. V1 supports controlled hardware tiers: 8 GB, 12 GB, 16+ GB VRAM.
4. V1 does not claim universal product-photography coverage.
5. Any case outside declared scope must be rejected or labeled "usable with revision".

## Frozen Exclusions
- Jewelry complex detail fidelity.
- Fashion on-body.
- Difficult transparency scenarios.
- Advanced mirror-grade reflections as primary commercial promise.
- Packaging where microtext must be perfectly preserved.
- Complex lifestyle scene storytelling.

## Scope Change Policy
Scope changes are blocked until these are complete:
- Main workflow stable.
- 10-case QA execution complete.
- Dependencies/version matrix frozen.
- Install + troubleshooting docs consistent with real behavior.
